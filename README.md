# 批量出图

本地 Web 工具：读取文件夹图片，按最多 10 个提示词批量调用 Noova 出图 API，并按提示词归档输出。

## 功能

- 设置页填写 API Key / Base URL（Key 明文仅保存在本地，接口默认不回传原文）
- 模型下拉 + 自定义模型（切换下拉会清空自定义，避免卡住旧模型）
- 全局源图/输出目录，支持系统文件夹选择
- 提示词卡片：可单独指定源图/输出；不填则走全局
- 输出子文件夹名自动由提示词截断 + hash 生成，避免冲突
- 输出文件重名自动加序号，不覆盖
- 同时只允许 1 个批次运行
- 任务进度会持久化到 `data/jobs.json`（重启后可查看历史；进行中任务会标记中断）

## 使用

1. 双击 `start.bat`
2. 打开页面后先在 **设置** 填 API Key
3. 选择源图目录、输出目录
4. 添加 1~10 个提示词
5. 开始批量出图

手动启动：

```bash
python -m pip install -r requirements.txt
python run.py
```

## 输出规则

- 未指定卡片输出目录时：`全局输出/提示词截断名_hash/`
- 文件名：`原图名.ext`；若已存在则 `原图名_2.ext`

## 说明

- 取消是本地软取消：不会撤回已提交到远端的任务
- 服务默认监听 `127.0.0.1`
- 配置文件：`data/settings.json`
- 任务记录：`data/jobs.json`
